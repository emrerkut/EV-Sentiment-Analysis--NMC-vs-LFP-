Our Methodology

From raw social media discussions to structured predictive insights.

1. Data Acquisition

We scraped real-world user discussions from top EV subreddits, filtering out noise to maintain high signal quality.

2. LLM Extraction

Using Groq (Llama-3) and Google Gemini, we transformed unstructured text into structured categorical features (e.g., Battery Type, Climate).

3. ML Prediction

Trained multiple classifiers (KNN, Decision Trees) handling class imbalance to predict user sentiment and find hidden patterns.
